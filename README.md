## Variables

###Assign a variable like so:

```
str = 12
bronze = "Josh sure is bronze!"
array = [1, 2, 3, 4, 5]
```

###You can use define variables to use them later in your code

```
str = 12
rts = 32

return str + rts
return str * rts
return rts / str
return str - rts
```

###There are many types of variables, such as:

* Integers
* String
* Arrays
